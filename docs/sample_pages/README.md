## 使用目的

これらのサンプルページは以下の目的で使用されます：

1. **DOM構造の解析**: ページ構造を理解してスクレイピングロジックを設計

3. **エッジケースの検証**: 様々なコメント形式に対応できるか確認
4. **ユニットテスト**: オフラインでのテストデータとして活用

## 開発者向けノート

### サニタイズ処理の制限事項

- **画像URL**: 100箇所以上存在するため、すべての削除は実施していません（構造確認用に残存）
- **md5ハッシュ**: 200箇所以上存在し、DOM構造に必要なため残存（通報機能のトリガーとして使用）
- **外部リンク**: ニュース記事等へのリンクは構造確認用に残存

### 追加のサニタイズが必要な場合

本番環境で使用する際は、以下の追加処理を検討してください：

```javascript
// 例: すべての外部画像URLを削除
document.querySelectorAll('img[data-src]').forEach(img => {
  img.removeAttribute('data-src');
});
```
